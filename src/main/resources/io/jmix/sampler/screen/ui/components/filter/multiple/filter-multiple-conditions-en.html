The Filter component enables quick data filtering by arbitrary conditions, as well as creating configurations for repeated use. Configuration can be created in design-time (DesignTimeConfiguration) or in run-time (RunTimeConfiguration). Design-time configurations cannot be changed, but can be copied in run-time using the Copy action from the drop-down menu of filter settings.

In this example, the configuration was created in design-time in XML, made default using default XML attribute. So it appears right after opening the screen.

You can add conditions to this configuration by copying it, or reset it by selecting Reset filter in the drop-down menu next to theRefresh button and create a new configuration. New conditions can be saved for future use by clicking Save, Save with values or Save as in the settings drop-down menu on the right.
