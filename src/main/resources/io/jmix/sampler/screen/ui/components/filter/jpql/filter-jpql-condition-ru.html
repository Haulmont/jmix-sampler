В данном примере конфигурация была создана во время разработки и установлена "по умолчанию" с помощью атрибута default. Поэтому конфигурация появляется сразу после открытия экрана.

Пользователи могут создавать условия на основе выражений JPQL, выбрав пункт Создать Jpql условие диалога Добавить условие поиска. Jpql условия также могут быть определены в XML экрана (jpqlFilter).

В данном примере JPQL условие позволяет пользователю фильтровать заказы по продукту, содержащемуся в строке заказа. Свойства условия можно увидеть в расширенном редакторе фильтра, доступном через пунктРедактировать меню настроек фильтра.
