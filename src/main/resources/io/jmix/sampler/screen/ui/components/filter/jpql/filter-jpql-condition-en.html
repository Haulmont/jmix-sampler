In this example, the configuration was created in design-time in XML, made default using default XML attribute. So it appears right after opening the screen.

A user can create conditions based on JPQL expressions by selecting Create Jpql condition in the Add search condition dialog. Jpql conditions can also be defined in the screen XML (jpqlFilter).

In this example, the Jpql condition allows a user to filter orders by a product contained in order lines. You can see the condition properties in the advanced filter editor available via the Edit settings menu item.
