The list of TwinColumn options can be specified using the following methods:

  • setOptionsList() - allows you to specify programmatically a list of component options;

  • setOptionsMap() - allows you to specify a string caption for each option value explicitly;

  • setOptionsEnum() - takes a class of enumeration as a parameter. The options list will consist of localized names of enum values, the value of the component will be an enum value.
