EntitySuggestionField combines the functionality of a suggestion field and a EntityPicker component, designed to display an entity instance. First way to link the component with an entity attribute is to specify dataContainer and property XML attributes.

query - an optional element which enables defining a query for selecting suggested values.

The search logic is defined with the help of SearchExecutor interface. The following parameters can be set:

  • asyncSearchDelayMs - the delay between the last key press action and async search;
  • suggestionsLimit - the limit of suggestions to be shown;
  • minSearchStringLength - min string length required to perform suggestions search.
