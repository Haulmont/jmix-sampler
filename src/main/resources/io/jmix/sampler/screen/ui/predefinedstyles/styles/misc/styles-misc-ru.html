Поле для ввода даты и времени имеет один предопределённый стиль - borderless

Компонент ProgressBar имеет один предопределённый стиль - point

Компонент Accordion имеет один предопределённый стиль - borderless

Вы можете применить эти стили в XML-дескрипторе используя атрибут stylename или в контроллере экрана используя константы из класса HaloTheme.
