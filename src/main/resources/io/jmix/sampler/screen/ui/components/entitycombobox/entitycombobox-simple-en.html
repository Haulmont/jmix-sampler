The EntityComboBox component allows a user to select an entity instance from a drop-down list and perform actions by pressing buttons on the right.

In fact EntityComboBox is a hybrid of ComboBox and EntityPicker. Thus it has the same features except the default list of actions added when determining the component in XML: for EntityComboBox these are entity_lookup and entity_open.

In this example, the first field has default actions, while the second one has entity_lookup and entity_clear actions specified explicitly.
