In this example, the configuration was created in design-time in XML, made default using default XML attribute. So it appears right after opening the screen.

The filter configuration conditions are contained in the tree. All top-level conditions are joined by logical condition (the default is AND). You can add AND or OR groups to create complex conditions.

A user can create group conditions by selecting Create Group condition in the Add search condition dialog. Group conditions can also be defined in the screen XML (groupFilter).

You can open the advanced conditions editor by selecting Edit in the settings drop-down menu.
