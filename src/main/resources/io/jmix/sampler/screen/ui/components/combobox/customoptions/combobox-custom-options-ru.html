Список опций компонента может быть задан произвольно с помощью следующих методов:

  • setOptionsList() позволяет программно задать список опций компонента;

  • setOptionsMap() позволяет явно задать строковые названия значениям опций;

  • setOptionsEnum() принимает в качестве параметра класс перечисления. Список опций будет состоять из локализованных названий значений перечисления, значением компонента будет являться выбранное значение перечисления.

Атрибуты nullName и inputPrompt позволяют задать строку, отображаемую вместо значения null.
