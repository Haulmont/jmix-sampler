ButtonsPanel - контейнер, унифицирующий использование и размещение компонентов (чаще всего кнопок) для управления данными в таблице.

Элемент buttonsPanel можно разместить как внутри таких компонентов как table, dataGrid и tree, так и в произвольном месте экрана.

Если buttonsPanel находится внутри table, то она комбинируется с компонентом pagination таблицы, тем самым оптимально расходуя место по вертикали. Кроме того, если экран открыт как lookup (например, из компонента EntityPicker) панель кнопок скрывается.
