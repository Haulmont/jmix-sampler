Компонент EntitySuggestionField объединяет в себе функциональность поля с подсказками и компонента EntityPicker, предназначенного для отображения экземпляра сущности. Первый способ связать данные с компонентом - это связать его с атрибутом сущности путем указания XML-атрибутов dataContainer и property.

query - необязательный элемент, который позволяет задать запрос для предлагаемых значений.

Логика поиска задаётся с помощью интерфейса SearchExecutor. Можно установить следующие параметры:

  • asyncSearchDelayMs - задержка между последней нажатой клавишей и запуском поиска;
  • suggestionsLimit - ограничение количества показываемых подсказок;
  • minSearchStringLength - минимальное количество символов, которое нужно ввести, чтобы запустить.
