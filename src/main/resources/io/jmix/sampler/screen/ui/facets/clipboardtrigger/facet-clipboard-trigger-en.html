ClipboardTrigger is a non-visual component which allows a user to copy text from a field to the clipboard.

ClipboardTrigger is described using the clipboardTrigger element.

  • input - identifier of a text field, must be a subclass of TextInputField: TextField, TextArea and so on.
  • button - identifier of a Button which triggers the copying.

The ClipboardTrigger.CopyEvent occurs when the copy button is clicked.
