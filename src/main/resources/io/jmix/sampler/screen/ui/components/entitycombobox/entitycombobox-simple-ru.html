Компонент EntityComboBox позволяет выбирать экземпляр сущности в выпадающем списке и выполнять действия нажатием на кнопки справа.

EntityComboBox является по сути гибридом ComboBox и EntityPicker, поэтому все описанное для этих компонентов верно и для него. Исключением является список действий по умолчанию, добавляемых при определении компонента в XML: для EntityComboBox это действия entity_lookup и entity_open.

В данном примере первое поле имеет действия по умолчанию, а второе - явно заданные entity_lookup и entity_clear.
