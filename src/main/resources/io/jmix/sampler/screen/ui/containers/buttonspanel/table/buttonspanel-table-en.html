ButtonsPanel is a container that streamlines the use and placement of the components (usually, buttons) for data management in a table.

buttonsPanel element can be located either inside a table, dataGrid, tree, or in any other place of a screen.

If the buttonsPanel is located in a table, it is combined with the table's pagination component thus using vertical space more effectively. Additionally, if a screen is opened as lookup (for example, from the EntityPicker component) the buttons panel becomes hidden.
