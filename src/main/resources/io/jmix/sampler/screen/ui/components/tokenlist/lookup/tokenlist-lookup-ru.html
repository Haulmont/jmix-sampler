Атрибуты элемента lookup:

  • Атрибут lookup задает возможность выбора значений через экран выбора сущностей.

  • Атрибут lookupScreen задает идентификатор экрана для выбора значений в режиме lookup="true". Если данный атрибут не задан, то идентификатор экрана выбора определяется по правилам, описанным для стандартного действия entity_lookup.

  • Атрибут openType можно задать способ открытия экрана выбора, аналогично описанному для стандартного действия entity_lookup. По умолчанию - THIS_TAB.
