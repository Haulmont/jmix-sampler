Компонент label может содержать вложенный элемент форматтера для представления значения атрибута в виде, отличном от стандартного для данного Datatype.

Виды форматтеров:

  • classNameFormatter - форматирует строку, содержащую путь к классу, где пакеты разделены точками, в строку, которая является именем класса.

  • collectionFormatter - форматирует коллекцию в строку, в которой элементы коллекции разделяются запятыми.

  • customFormatter - устанавливает пользовательский форматтер для преобразования значения компонента в строку.

  • dateFormatter - форматирует значение даты в строку в зависимости от строки формата.

  • numberFormatter - форматирует числовое значение в строку в зависимости от строки формата.
