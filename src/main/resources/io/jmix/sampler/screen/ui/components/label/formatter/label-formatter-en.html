Label component may contain a nested formatter elements that allows to represent the attribute value in a format different from the standard for this Datatype.

Formatter elements:

  • classNameFormatter - formats the string that contains the class path, where the packages are separated by dots, into a string that is the class name.

  • collectionFormatter - formats collection into a string where the elements of the collection are separated by commas.

  • customFormatter - sets a custom formatter to format the component value to a string.

  • dateFormatter - formats the Date value into a string depending on the format string

  • numberFormatter - formats the Number value into a string depending on the format string.
