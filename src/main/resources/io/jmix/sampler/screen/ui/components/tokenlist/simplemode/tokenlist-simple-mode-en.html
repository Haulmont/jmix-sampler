When simple XML attribute of a TokenList is set to true, the items selection component will be hidden with only the Add button left. Clicking the Add button opens the screen with the list of entity instances which type is defined by the dataContainer. Selection screen identifier is selected according to the rules for the entity_lookup standard action.

If the value of the multiselect attribute is set to true, then true will be passed to parameters map of the lookup screen for the MULTI_SELECT key. In this example, this parameter is used to switch the lookup table to multi-select mode.
